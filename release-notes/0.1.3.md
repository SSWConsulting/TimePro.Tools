# 0.1.3 - 2026-06-22

## Added

- Added the first installable GitHub Release package for the `tp` global tool.
- Added version-stamped `tp --version` output with package version and build commit.
- Added the one-shot macOS/Linux and Windows install scripts.
- Added leave-aware weekly coverage checks, including public holiday and approved leave coverage.
- Added MCP support for week coverage and leave-balance lookups.
- Added `tp project recent` to surface likely projects before entering timesheets.
- Added daily scrum generation improvements, including smarter item selection and configurable templates.

## Fixed

- Improved JSON error handling so `--json` failures stay parseable on stdout.
- Corrected several TimePro API request and serialization shapes.
- Improved cloud setup and release workflow reliability.
